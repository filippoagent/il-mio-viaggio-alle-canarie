# Il mio viaggio alle Canarie

Sito React + TypeScript per un'agenzia a Tenerife che propone noleggio motorini, escursioni, camere e pacchetti viaggio.

## Sviluppo

```bash
npm install
npm run dev
```

Il sito viene avviato su `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

L'output di produzione viene generato nella cartella `dist`.

## Deploy

Piattaforme consigliate:

- Vercel: molto rapida per React/Vite, preview automatiche e collaborazione.
- Netlify: ottima alternativa, semplice per siti statici e form.
